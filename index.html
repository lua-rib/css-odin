Desenvolvimento front-end

De forma geral, o Front-end compreende a parte visual de sites e aplicações. Ou seja, a área das páginas em que as pessoas podem interagir. Isso significa que o Front-end se relaciona com as partes dos sites que mais se aproximam da pessoa usuária. Sendo assim, as pessoas que trabalham com Front-end são responsáveis por desenvolver uma interface gráfica através de tecnologias bases da Web: HTML, CSS e JavaScript.

Principais tecnologias

Seu nome tem origem do inglês Hyper Text Markup Language, que significa “linguagem de marcação de hipertexto”.

Cascading Style Sheet, mais conhecido como CSS, é uma linguagem de estilos que, assim como o HTML, também não é considerada uma linguagem de programação. Ela é responsável por separar a parte estrutural da aplicação (que ficará nas mãos do HTML) da parte estética.

Ele adiciona movimento às páginas web, além de permitir o processamento e transformação de dados enviados e recebidos. Ele permite criar conteúdos que se atualizam de forma dinâmica e animada, dando vida àsaplicações que antes eram apenas estruturadas com HTML de forma estática.

React é um framework JavaScript criado pelo Facebook (atual Meta) que é usado para criar interfaces de usuário (UI) em aplicativos web. Ele é popular por ser fácil de usar, altamente flexível e escalável, e é usado por muitas empresas de tecnologia, incluindo o Facebook, Instagram e Airbnb.

Front-end é tudo o que você vê e interage em um site ou aplicativo. Quem trabalha nessa área cria o visual, os botões, as animações e até a forma como as coisas funcionam quando você clica.
